# Instrucciones para agregar el logo SVG

## Ubicación del archivo:
- Nombre: `logo.svg`
- Ubicación: Raíz del repositorio (mismo nivel que el HTML)

## Especificaciones del SVG:
- Dimensiones recomendadas: 50x50px o proporcional
- Formato: SVG optimizado
- Colores principales: #2d5016 (verde) y #ffa500 (naranja)

## Después de subir logo.svg:
El logo aparecerá automáticamente con la animación buildUp al cargar la página.
